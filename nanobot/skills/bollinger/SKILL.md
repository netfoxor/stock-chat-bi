# 布林带信号检测 Skill

当用户问"**布林带**"、"**超买/超卖**"、"**突破上/下轨**"、"**均值回归信号**"等问题时使用。

## 运行（严格按此格式，否则会踩 Windows shell 引号陷阱）

用 `exec` 工具执行：

```
python skills\bollinger\scripts\detect.py --ts-code 600519.SH --start 2024-01-01 --end 2024-12-31
```

**三条硬规定**（违反必挂）：

1. **不传 `working_dir`** —— `exec` 默认 cwd 就是 `nanobot/`，相对路径可直达
2. **用相对路径** `skills\bollinger\scripts\detect.py`，**不要**写绝对路径
3. **整个 command 里不加任何引号**（路径不含空格，加引号反而触发 `\"` 解析错乱）

参数：

- `--ts-code`（必填）：Tushare 代码
- `--start`（可选）：起始交易日，格式 `YYYY-MM-DD`。**省略则默认最近一年**
- `--end`（可选）：结束交易日，格式 `YYYY-MM-DD`。**省略则为今天**

区间规则（与脚本内实现一致）：

| 参数组合                   | 实际区间                          |
| -------------------------- | --------------------------------- |
| 都不给                     | 近一年 → 今天                      |
| 仅 `--start`               | start → 今天                       |
| 仅 `--end`                 | (end - 1 年) → end                 |
| 都给                       | start → end（end 超今天自动回退）  |

输出约定：

- **stdout** 是完整 markdown：信号摘要 + 明细表（若有信号）+ 图表占位 `![布林带](chart:charts/boll_xxx.json)`
- **必须原样转发给用户**
- 非零 exit code：把原因告诉用户

## 规则

- 窗口 20 日，`upper = MA20 + 2σ`、`lower = MA20 - 2σ`
- **超买**：收盘价 > upper
- **超卖**：收盘价 < lower
- 区间内交易日数不足 25 条会报错

## 出错自愈清单（Self-Heal）

| 症状                                                   | 处方                                                             |
| ------------------------------------------------------ | ---------------------------------------------------------------- |
| `can't open file '...scripts\"...\"'` / `Errno 22`      | 触犯了引号规则。去掉 `working_dir` + 去掉所有引号，重试一次       |
| `No such file or directory ... detect.py`               | 路径写错，按模板 `skills\bollinger\scripts\detect.py` 再试        |
| `错误：数据库文件未找到`                                | 数据缺失，**不要重试**，如实告诉用户                              |
| `错误：未找到 XXX 在 [...] 的日线数据`                   | 该股在该区间无行情，换区间或换代码再试                           |
| `错误：仅有 N 条日线，不足 25 条`                        | 区间太短，放宽 `--start` 到更早一点再试                          |
| `错误：start_date / end_date 须为 YYYY-MM-DD`           | 参数格式修正后重试                                               |

> ⚠️ 同一个错误已经试过 2 次就停手，**不要无限重试**。

## 解读要点

- 布林带**频繁触碰上/下轨**说明波动放大，并不必然等于反转信号
- 突破上轨后如果**连续站稳**，可能是强势延续；跌破下轨后快速回弹，常见均值回归
- 窄口收敛（带宽接近历史低位）后发生"布林带突破"，通常波动率进入扩张期
